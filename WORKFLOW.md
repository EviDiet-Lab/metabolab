# MetaboLab 運用マニュアル

## 記事制作ワークフロー（週次）

### ステップ1：トレンドリサーチ（毎週月曜）

以下のツールで調査を実行します：

- **Googleトレンド**: `https://trends.google.co.jp/trends/`
  - キーワード: メタボ, ダイエット, 糖質制限, 血糖値, 中年太り
- **YouTube**: メタボ / 健康 / 筋トレ関連の人気動画をチェック
- **X (Twitter)**: 健康・メタボ関連トレンドワードを収集

出力例：
```text
【MetaboLab】今週のトレンドレポート（YYYY/MM/DD）
- 急上昇キーワードTOP3：○○、△△、□□
- YouTube人気テーマ：○○
- Xでの話題：△△
- 推奨記事テーマ：□□
```

### ステップ2：エビデンス（論文）リサーチ

ステップ1で特定したテーマに関連する論文を検索：

- **PubMed**: `https://pubmed.ncbi.nlm.nih.gov/`
- **Google Scholar**: `https://scholar.google.co.jp/`

選定基準：
- 査読付き学術誌に掲載
- 可能な限りオープンアクセス
- 発表年が直近5年以内
- Impact Factorが高いジャーナル優先

出力例：
```text
【MetaboLab】今週のエビデンスレポート（YYYY/MM/DD）
- トレンドテーマ：○○
- 選定論文1：タイトル / 著者 / ジャーナル / 年 / DOI
- 論文の要点（3〜5箇条書き）
- 実生活への応用ポイント
```

### ステップ3：記事執筆

`src/content/articles/` に Markdown ファイルを作成。

## 更新スケジュール

- 新規記事公開: 週3本（月・水・金）
- 既存記事のアップデート: 月1回
- トレンドリサーチ: 毎週月曜
- 論文リサーチ: 毎週火曜
- 記事執筆: 毎週水〜金曜
- レビュー（PR作成）: 公開前日

## 法的規制（厳守）

以下の表現は**絶対に使用しない**：

- 「〇〇を食べれば痩せる」「△△で確実に効果」などの誇大表現
- 医薬品的な効果効能（「脂肪を分解する」「血糖値を下げる」など）
- 個人の体験談を一般的事実として提示

エビデンスを主張する場合は、必ず論文のDOIリンクまたはPubMed IDを明記すること。
